# Portfolio

A simple portfolio website for a designer/developer. Built with Next.js 14, TypeScript, and Tailwind CSS.

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deploy on Vercel

1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) and import your repository
3. Vercel will auto-detect Next.js—click Deploy

That's it. Your portfolio will be live with automatic deployments on every push.

## Customize

- **Name & branding**: Update "Your Name" in `app/page.tsx` (nav + footer)
- **Projects**: Replace the `projects` array with your real work—add images, links, descriptions
- **About**: Edit the about section with your bio, location, skills, and interests
- **Contact**: Swap in your real email and social links
- **Colors**: Adjust `tailwind.config.ts` (cream, ink, muted, accent)
